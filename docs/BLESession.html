BLESession

BLESession

new BLESession(runtime, deviceOptions, connectCallback)

A BLE device session object. It handles connecting, over web sockets, to BLE devices, and reading and writing data to them.

Parameters:
Name Type Description
runtime Runtime

the Runtime for sending/receiving GUI update events.
deviceOptions object

the list of options for device discovery.
connectCallback object

a callback for connection.

Methods

connectDevice(id)

Try connecting to the input peripheral id, and then call the connect callback if connection is successful.

Parameters:
Name Type Description
id number

the id of the peripheral to connect to
